Introduction to HTML and CSS

Welcome to the world of web development! If you've ever wondered how websites are created or wanted to build one yourself, understanding HTML and CSS is essential. These two technologies form the backbone of the web, allowing you to structure content and style it to create engaging experiences.

Basic HTML Document Structure

Every web page starts with HTML. It's the standard markup language for creating web pages and applications.


        <!DOCTYPE html>
        <html>
          <head>
            <title>My First Web Page</title>
          </head>
          <body>
            <h1>Hello, World!</h1>
          </body>
        </html>

Breaking It Down:

  • <!DOCTYPE html> declares the document type as HTML5.
  • <html> wraps all the content on the page.
  • <head> contains meta-information like the page title.
  • <body> holds the content that displays in the browser window.
Common HTML Elements

HTML uses elements to define the structure and content of a webpage.

Headings and Paragraphs

Headings (<h1> to <h6>) are used for titles and subtitles.


        <h1>Main Title</h1>
        <h2>Subheading</h2>
        <p>This is a paragraph providing more details.</p>

Lists

Lists organize content into bullet points or numbered items.

  • Ordered List:

        <ol>
          <li>First item</li>
          <li>Second item</li>
        </ol>
  • Unordered List:

        <ul>
          <li>Bullet point one</li>
          <li>Bullet point two</li>
        </ul>

Links and Images

Add hyperlinks and images to enrich your content.


        <a href="https://www.example.com">Visit Example.com</a>
        <img src="image.jpg" alt="Descriptive text">
Introduction to CSS

Cascading Style Sheets (CSS) bring life to your HTML by controlling the layout and appearance.


        body {
          background-color: #f0f0f0;
          font-family: Arial, sans-serif;
        }

Key Concepts:

  • Selectors target HTML elements.
  • Properties define what you want to change.
  • Values specify the settings for those properties.
Linking CSS to Your HTML

There are several ways to apply CSS to HTML.

External Stylesheet

Link a CSS file using the <link> tag inside <head>.


        <head>
          <link rel="stylesheet" href="styles.css">
        </head>

Internal Stylesheet

Include CSS directly within your HTML file using <style> tags.


        <head>
          <style>
            h1 {
              color: navy;
            }
          </style>
        </head>

Inline Styles

Apply styles directly to an element using the style attribute.


        <p style="color: gray;">This is an inline styled paragraph.</p>
CSS Selectors and Properties

Selectors are patterns used to select the elements you want to style.

Basic Selectors

  • Element Selector: Targets all instances of an element.

        p {
          font-size: 16px;
        }
  • Class Selector: Targets elements with a specific class attribute.

        .highlight {
          background-color: yellow;
        }
  • ID Selector: Targets a unique element with a specific ID.

        #header {
          text-align: center;
        }

Combining Selectors

Apply styles more precisely.


        ul li {
          list-style-type: square;
        }
The CSS Box Model

Understanding the box model is crucial for layout and design.

Components:

  • Content: The text or images in the box.
  • Padding: Space around the content.
  • Border: A line surrounding the padding and content.
  • Margin: Space outside the border.

        div {
          margin: 20px;
          padding: 15px;
          border: 2px solid black;
        }
Positioning Elements with CSS

Control the placement of elements on your page.

Position Properties

  • Static: Default positioning.
  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport.

        .footer {
          position: fixed;
          bottom: 0;
          width: 100%;
        }

Floating Elements

Float elements to align them side by side.


        .left {
          float: left;
          width: 50%;
        }
        .right {
          float: right;
          width: 50%;
        }
Responsive Design with Media Queries

Create websites that look great on any device


        @media (max-width: 600px) {
            body {
              background-color: lightblue;
            }
            .menu {
              display: none;
            }
        }

Why Use Media Queries?

  • Adapt layout to different screen sizes.
  • Hide or show content based on device capabilities.
  • Enhance user experience across devices.
Styling Text and Fonts

Make your typography stand out.

Font Properties


        h1 {
          font-family: 'Roboto', sans-serif;
          font-size: 2em;
          font-weight: bold;
          color: #333;
        }

Text Decoration and Transformation


        a {
          text-decoration: none;
        }
        p.uppercase {
          text-transform: uppercase;
        }
Working with Colors in CSS

Colors can be specified in various ways.

Color Notations

  • Hexadecimal:

        .header {
          background-color: #ff5733;
        }
  • RGB and RGBA:

        .overlay {
          background-color: rgba(0, 0, 0, 0.5);
        }
  • HSL and HSLA:

        .button {
          color: hsl(120, 100%, 50%);
        }
Using CSS Flexbox and Grid

Modern layout modules for complex designs.

Flexbox

Arrange elements in a flexible way.


        .container {
          display: flex;
          justify-content: space-between;
        }

Grid

Create two-dimensional layouts.


        .grid {
          display: grid;

          grid-template-columns: repeat(3, 1fr);
        }
Adding Transitions and Animations

Enhance interactivity with smooth effects.

Transitions


        .button {
          background-color: blue;
          transition: background-color 0.3s ease;
        }
        .button:hover {
          background-color: green;
        }

Keyframe Animations


        @keyframes fadeIn {
          from {
            opacity: 0;
          }
          to {
            opacity: 1;>
          }
        }
        .modal {
          animation: fadeIn 1s forwards;

        }
